## Usage

After running it will display the bord and the turn.
Type 2 numbers seperated by a comma to represent the co-ordinate where you want to take your turn.
The possible co-ordinates are below:

1, 1 | 2, 1 | 3, 1

1, 2 | 2, 2 | 3, 2

1, 3 | 2, 3 | 3, 3
